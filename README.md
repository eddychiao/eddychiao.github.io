# Eddy Chiao — Personal Portfolio

Live site: [eddychiao.github.io](https://eddychiao.github.io/)

A personal portfolio and photography site built to showcase projects, experience, and photography work.

## Pages

- **Home**
- **About** — Personal bio, tech stack, and work experience timeline
- **Projects** — List of projects I've worked on and their details _(in progress)_
- **Photography** — Redirects to my photography website
- **Contact** — Links to my email, LinkedIn, GitHub, and Instagram

## Features

- 9 switchable color themes with localStorage persistence
- Typing animations on page titles
- Responsive layout across screen sizes

## Tech Stack

- React 19 + TypeScript
- React Router v7
- Material-UI, Ant Design, Bootstrap
- Tailwind CSS
- Deployed via GitHub Pages

## Running Locally

```bash
npm install
npm start   # http://localhost:3000
npm run build  # production build → /build
```
